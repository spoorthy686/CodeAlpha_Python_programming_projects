# Smart Automation Hub

## Project Overview

Smart Automation Hub is an advanced Python-based desktop application designed to automate repetitive file management tasks through an interactive graphical interface. The application enhances basic automation concepts by providing smart organization, duplicate detection, and file analysis features.

This project was developed as part of the CodeAlpha Python Programming Internship with additional enhancements beyond the original requirements.

---

## Features

### Automation Features
- Folder selection system
- Automatic file organization
- Duplicate file detection
- Email extraction and identification
- Real-time processing

### Dashboard Features
- Organized results display
- Statistics summary
- Success notifications
- Dynamic activity updates

### Smart Features
- Quote of the Day
- File analysis
- Duplicate count display
- Email count tracking

### User Interface Features
- Modern graphical interface
- Clean dashboard layout
- Interactive controls
- User-friendly design

## How to Run

Clone the repository:

```bash
git clone <repository-link>
```

Navigate to the project folder:

```bash
cd SmartAutomationHub
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
